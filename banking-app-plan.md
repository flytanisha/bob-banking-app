# Banking Web Application — Plan

**Stack:** Python 3 · Flask · SQLite · Jinja2 · Bootstrap 5 (CDN)  
**Folder layout:** `BACKEND/` for all Python; `FRONTEND/templates/` for HTML; `FRONTEND/static/` for CSS  
**Entry point:** `python BACKEND/app.py`  
**No external services** — everything runs locally from a single command.

See [IMPLEMENTATION_PLAN.md](IMPLEMENTATION_PLAN.md) and [STEP_BY_STEP_IMPLEMENTATION_GUIDE.md](STEP_BY_STEP_IMPLEMENTATION_GUIDE.md) for full details.
